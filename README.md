# Merch Pet land

`Merch` open sources store for pet, building with ‚ù§ Next.js

## Fork and Clone repository

```sh
git clone https://github.com/castrogarciajs/merch
```

## Package manager

we use `pnpm` if you do not have pnpm, install it with

```sh
npm install -g pnpm
```

## Development

Getting started for merch,

```sh
pnpm install
```

```sh
pnpm dev
```

## Lint and Test

Run test and Lint for mech

we use `eslint` look at the rules in file `.eslintrc`

```sh
pnpm lint
```

For testing use `vitest`

Test start

```sh
pnpm test
```

Test dev

```sh
pnpm test.dev
```

Test ui

```sh
pnpm test.ui
```

Test coverage

```sh
pnpm test.coverage
```
